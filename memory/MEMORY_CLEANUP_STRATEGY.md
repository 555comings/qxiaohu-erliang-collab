## 1. 策略目标

处理历史文件中的问题：
- 空文件
- 乱码文件
- 低质量内容

---

## 2. 边界定义

| 类型 | 边界 | 处理方式 |
|------|------|---------|
| 空文件 | < 50 bytes | 检查是否有效 |
| 小文件 | < 200 bytes | 检查内容价值 |
| 乱码文件 | 文件名/内容乱码 | 冻结，不自动修复 |
| 低质量 | 重复内容 > 80% | 评估后处理 |

---

## 3. 分级标准

### A类（可处理）
- 内容正常
- 无引用
- 低风险
- 可后续改名

### B类（需审查）
- 内容正常
- 可能被引用
- 需先查引用

### C类（冻结）
- 内容损坏
- 文件名/内容乱码
- 不自动修复
